## 系统配置要求（System Requirements）

好消息是：目前的 Demo 对配置要求不算高。

根据公开的配置说明和玩家实测，大致可以按下面这个标准来预期：

### 最低配置（Demo / 早期版本参考）

- **系统：** Windows 10（64 位）  
- **CPU：** Intel Core i3-2100 或类似双核  
- **内存：** 4 GB  
- **显卡：** 大约 **GTX 750 Ti / RX 460** 等级，2 GB 显存  
- **硬盘：** 几个 G 空间足够  
- **网络：** 宽带（联机 / Steam 必备）

如果你电脑这几年还能跑其他独立恐怖游戏，大概率没问题。

### 推荐配置

- **系统：** Windows 10/11（64 位）  
- **CPU：** Intel Core i5-6500 或同级  
- **内存：** 8 GB  
- **显卡：** GTX 1050 Ti / RX 560 或以上，4 GB 显存  
- **硬盘：** 多预留一点空间给更新和录制

如果你配置刚好卡在边缘，也没关系 —— 画质拉低一点，分辨率往下调一点，一样能玩。

## 安装步骤（Step-by-Step Installation）

我们分别说三种主要下载路径的安装流程。

### A. Itch.io Demo 安装

1. 打开 Shift At Midnight 的 **官方 Itch.io 页面**。  
2. 点击 **Download**（下载）按钮，选择 Windows 版本。  
3. 把 `.zip` 文件保存到一个好找的位置（例如：桌面 / 下载 / 自己建的 `Games` 目录）。  
4. 下载完成后：
   - 右键 ZIP → **“解压全部…”**（或用 7-Zip / WinRAR 解压）  
   - 选好你想放游戏的文件夹，比如：`C:\Games\ShiftAtMidnight`  
5. 解压后，打开该文件夹，找到：
   - `ShiftAtMidnight.exe` 或类似的启动文件  
6. 双击运行，就能进游戏。

如果这时 Windows 弹出“你确定要运行吗”，先别慌，往下看“安全提示”那一节。

### B. Steam Demo 安装

1. 打开 **Steam 客户端**。  
2. 在商店里搜索 **“Shift At Midnight”**。  
3. 进入商店页面：
   - 如果有 **“下载试玩”/“安装 Demo”** 按钮，点它；  
   - 顺便点个 **“加入愿望单（Wishlist）”**，以后更新好追。  
4. Steam 会问你装在哪个盘，选好后确认。  
5. 等下载 + 安装完。  
6. 在库里找到游戏，点击 **“开始游戏（Play）”**。

之后更新都是 Steam 自动帮你搞定，是最省心的方案。

### C. GitHub ZIP 镜像安装

如果你从 GitHub Release 下载了 Demo 压缩包：

1. 在 Releases 页面点 `.zip` 文件（例如 `ShiftAtMidnight.zip`）。  
2. 保存到你的电脑。  
3. 下载结束后：
   - 右键 ZIP → **“解压全部…”**  
   - 选一个安装目录，例如：`D:\Games\ShiftAtMidnight_Demo`  
4. 解压完打开文件夹，找到 EXE（例如 `ShiftAtMidnight.exe`）。  
5. 双击运行。

如果 Windows 提示“来自未知发布者”，这在小团队游戏里很常见，参考下一节。

## 第一次启动 & 安全弹窗处理（First Launch & Security Prompts）

在 Windows 上跑独立游戏，你多半会遇到那块经典蓝色提示：

> **“Windows 已保护你的电脑”**

这是 **Windows SmartScreen** 在说：“这文件我不认识”。  
对于没有大公司证书签名的小 indie 游戏、GitHub 压缩包镜像，这很常见。

### 遇到 “Windows 已保护你的电脑” 怎么办？

1. 在蓝色 SmartScreen 窗口上，先点击 **“更多信息（More info）”**。  
2. 看一下细节：
   - 文件名对不对  
   - 发布者可能会显示 “Unknown”（未知发布者）  
3. 如果你**确定**是从官方 Itch.io / Steam / 可信的 GitHub 链接下的：
   - 点击 **“仍要运行（Run anyway）”**。

如果你不确定这文件从哪来的，或者是别人私发给你的可疑链接，建议：  
**直接删掉，重新去官方页面下载一份**，别拿自己电脑开玩笑。

### 杀毒软件 / Windows Defender 警告

有时候杀毒或 Defender 会：

- 把 EXE 直接隔离  
- 或者弹个提示吓你一下

独立游戏没啥签名、没啥“信誉记录”的情况下，这种误报也挺常见。

如果你确认自己下的是正版 Demo：

1. 打开 **Windows 安全中心 / 杀毒软件**。  
2. 找到 **“保护历史 / Quarantine（隔离区）”** 之类的页面。  
3. 找到被拦截的那个 EXE。  
4. 选择 **“允许 / 恢复 / 信任”**。  
5. 如有必要，把游戏所在文件夹加进 **排除 / 白名单**。

但如果杀毒软件报的是实打实的木马名、而且你不确定来源——  
那就听杀毒的，删干净，然后老老实实上 Itch.io / Steam 重下。

## 如何更新到新版本（How to Update to New Version）

Shift At Midnight 目前还在开发中，后续肯定会有新 Demo、新版本出现。

### 使用 Steam 更新

最简单粗暴的一种：

- Steam 默认会自动更新游戏。  
- 如果觉得自己版本不对劲：
  1. 在库里右键游戏 → **“属性”**  
  2. 找到 **“已安装文件（Installed Files）”**  
  3. 点 **“验证游戏文件完整性”**  
  4. 交给 Steam 自己修。

### 使用 Itch.io 更新

在 Itch.io 上：

1. 看一下 Shift At Midnight 页面上有没有新版本 / 更新日期变化。  
2. 如果有新版本：
   - 重新下载最新的 ZIP。  
3. 建议解压到 **新的文件夹**，不要直接覆盖旧版本。  
4. 确认新版本运行正常后，再把旧版本删掉释放空间。

### 使用 GitHub Demo 镜像更新

如果你是从 GitHub 下的：

1. 去 Release 页面看 **版本号 / tag**，例如 `v1.00`、`v1.01`。  
2. 有新版本就：
   - 下新的 ZIP  
   - 解压到一个新目录  
3. 运行新版本 EXE 测一下。  
4. 没问题的话，把旧版本文件夹删了就行。

## 常见问题排查（Troubleshooting）

### 1. 双击没反应 / 一闪而过就崩了

**表现：**

- 双击 EXE 什么都没发生；  
- 或者窗口一闪就退。

**尝试：**

1. 右键游戏 → **以管理员身份运行**。  
2. 确保 Windows 更新打全。  
3. 更新你的 **显卡驱动**（NVIDIA / AMD / Intel）。  
4. 如果游戏提示缺少某些运行库（比如 Visual C++、.NET），从微软官网装一份。

### 2. 游戏打开了，但鼠标在游戏里不动

有玩家反馈过：游戏能打开，菜单里鼠标正常，但进游戏后视角不动——鼠标像失灵了一样。

可以试试：

- 在游戏窗口里点一下，然后按 `Alt + Enter` 切换 **窗口 / 全屏**。  
- 按 `Esc` 回到菜单，再返回游戏。  
- 去 **Controls** 看看鼠标视角是不是被改没了。  
- 关掉可能抢焦点的覆盖层：录屏软件、其它程序浮窗等。

实在不行，就重启一次游戏，或者重新解压一份干净版本。

### 3. 解压 ZIP 报错 / 解不出来

如果 ZIP 解压失败：

1. 直接删掉坏的 ZIP。  
2. 重新下载一次（网络不稳很容易搞坏压缩包）。  
3. 用 **7-Zip / WinRAR** 等工具解压。  
4. 偶尔 Windows 会把从网上下的文件“锁住”，可以：
   - 右键 ZIP → 属性  
   - 如果有“解除锁定（Unblock）”之类的选项，勾上再解压。

### 4. 杀毒老是删 EXE

如果你每次解压出来，EXE 都立刻被杀毒吃掉：

1. 打开杀毒 / Windows 安全中心。  
2. 找到哪一条记录把它干掉了。  
3. 如果你确认来源没问题，就标记为 **允许 / 信任 / 恢复**。  
4. 给游戏文件夹加一个 **排除项**，避免每次都被扫掉。

再强调一遍：  
一旦你对文件来源有怀疑，就别跟杀毒对着干，删了重下。

### 5. 卡得像 PPT

如果画面一直在 PPT 模式：

1. 打开 **设置 → 图像 / Graphics**。  
2. 优先拉低：
   - 阴影  
   - 后期特效  
   - 分辨率比例  
3. 关掉：
   - 动态模糊  
   - 景深（如果有）  
4. 关掉后台占资源的软件：浏览器一堆标签、各种启动器、录屏、流媒体等。

如果你刚好在最低配置边缘，那就：

- 分辨率先拉到 900p 或 720p  
- 所有特效走 Low  
- 追求的是“能玩”，不是截图。
